A simple 2-player space shooter game built using Python and Pygame.

## Features
- Two-player controls
- Bullet shooting
- Collision detection
- Health system
- Sound effects for shooting and hits
- Automatic game restart after 5 seconds

## Technologies Used
- Python
- Pygame

## How to Run
1. Install Python
2. Install pygame:
   pip install pygame
3. Run the game:
   python main.py

## Controls
### Player 1 (Yellow)
- Move: W / A / S / D
- Shoot: F key

### Player 2 (Red)
- Move: Arrow Keys
- Shoot: J key
